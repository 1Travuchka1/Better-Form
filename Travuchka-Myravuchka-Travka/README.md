# Travuchka-Myravuchka-Travka

Мульти-крокова форма реєстрації з валідацією даних.

## Опис проекту

Простий веб-додаток з двокроковою формою для збору інформації користувача:

- **Крок 1:** Введення ПІБ (ім'я, прізвище, по-батькові)
- **Крок 2:** Введення адреси та контактів (вулиця, номер будинку, телефон)

Форма включає валідацію на кожному кроці та можливість повернутись назад.

## Як запустити

1. Клонуйте репозиторій:

```bash
git clone https://github.com/ваш-username/travuchka-myravuchka-travka.git
```

2. Відкрийте `index.html` у браузері.

**Примітка:** Проект використовує ES6 модулі, тому для коректної роботи відкривайте через локальний сервер (Live Server в VS Code) або просто відкрийте файл у браузері.
